A page and widget what display all Categories with custom thumbnail images. It is look like a gallery. Wordpress is not support the thumbnail images for Categories, but this plugin is extend this deficiency. It is use a discrete effect for display and it is there more options.

== Description ==

Category List Portfolio Page is appear on a custom page by a shortcode - [clpp-belicza].
The Category list is not text-list. It is an animated list with thumbnail icons/images and titles like a Gallery or a Portfolio page.
You can setup the Category page or widget on Admin area: columns (1-4), Image of the Category, Each rounded corners on the images (0px-15px - CSS3), Width and height of the images
It is use jQuery from the WP core, so it is not interfering for other plugins.
It is a translation ready plugin.
There is the default English language and the Hungarian translating.

For more info: http://belicza.com/wordpress/category-list-portfolio-page/

A few notes about the sections above:

*   Require 2.8 Wordpress minimum

== Installation ==

1. Upload the plugin to your wp-content/plugins directory (or install it via the wordpress interface)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a new page: PAGES > NEW PAGE
4. Paste the shortcode into content of the page: [clpp-belicza]
5. Save and look the page.
6. Or activate the Category Icons widget
6. You can modify all options from this: PLUGINS > Category List Portfolio Page Options

== Frequently Asked Questions ==

= Can I turn off the animation? =
Yes. Delete the category-list-portfolio-page/scripts/animation.js

= I activated the plugin, but it is not work:S =
Create a new Page in your admin arena and past this code: [clpp-belicza]

= My icons is too small. =
On page: Modify the width and height to bigger.
In widget: Modify the height to bigger and then the image width will be auto-resized to bigger.
